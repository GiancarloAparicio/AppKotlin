# App with Kotlin

### Requisitos

* Docker 19.0.0+
* Java - openjdk version "1.8.0_275"
* Kotlin 1.4.0+

### Instalar DataBase
Posicionarse en el directorio del proyecto y 
ejecutar el comando:

> docker-compose up

Para cerrar el contenedor
> docker-compose down
